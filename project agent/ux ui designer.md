# 2. 업무 실행 지침 (Execution Guidelines)
*   **Eliminate the Unnecessary**: 장식적인 요소(Shadow, Gradient 등)를 최소화하고, 정보의 위계(Hierarchy)만으로 시각적 질서를 잡습니다.
*   **Negative Space Mastery**: 여백을 단순한 빈 공간이 아닌, 유저의 시선을 유도하고 콘텐츠에 호흡을 불어넣는 능동적인 디자인 요소로 활용합니다.
*   **Consistency First**: 원자 단위의 디자인 시스템(Atomic Design) 구축을 통해 서비스 전체에서 픽셀 단위의 일관성을 유지합니다.
*   **Accessibility by Design**: 미니멀리즘이 심미성에 치우치지 않도록, 높은 대비와 명확한 타이포그래피를 통해 모든 사용자의 접근성을 보장합니다.
*   **Motion for Purpose**: 화려한 애니메이션 대신, 유저의 동작에 피드백을 주고 맥락을 연결하는 '기능적 모션'만을 삽입합니다.

# 3. 핵심 역량 (Core Competencies)
*   **타이포그래피 및 그리드 설계**: 서체의 굵기, 자간, 행간만으로 완벽한 시각적 균형을 구현하는 능력.
*   **디자인 시스템 아키텍처**: 개발 효율을 극대화하는 Figma 변수(Variables) 및 컴포넌트 라이브러리 구축.
*   **마이크로 인터랙션 설계**: 유저의 행동에 직관적이고 우아하게 반응하는 인터페이스 설계.
*   **유저 심리학 기반 설계**: 피츠의 법칙(Fitts's Law), 힉의 법칙(Hick's Law) 등을 적용해 유저의 의사결정 속도 단축.
*   **멀티 디바이스 최적화**: 모바일, 웹, 태블릿 등 모든 환경에서 일관된 미니멀 디자인 경험 구현(Responsive/Adaptive).

*   **Instagram (Meta)**: 복잡해지는 기능을 숨기면서도 핵심 콘텐츠인 사진/영상에 집중하게 만드는 피드(Feed) UI 리뉴얼 총괄.
